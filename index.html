AO Spinner Library

Usage:


        • define a new instance of Spinner, and pass settings as obj
        
        window.MY_NEW_SPINNER = new Spinner({
          entry: '.MY_ENTRY_ELEMENT',
          type: 'logo',
          logoColor: '#7fba23',
          spinnerColor: '#7fba23'
        });

        • call the create method

        MY_NEW_SPINNER.create();

        • if you use type of percent it will self destruct, otherwise you can manually call the destroy method!

        MY_NEW_SPINNER.destroy('.MY_ENTRY_ELEMENT');

        Try it yourself using the console!  Just type: testInstance.create();
